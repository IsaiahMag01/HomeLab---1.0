# Removing ZSF Pool That has been created
- Kind of Silly thing but I did not understand when I was clearing out one of the disk that I partitioned that before you can wipe it first you must do the following
1. Enter the Node
2. Disk > ZFS
3. Select the pool you are trying to remove -> Hit the blades in the top right
4. Delete
5. Type ZFS Pool name
6. Go to Disk
7. Select Drive
8. Wipe -> Ok
